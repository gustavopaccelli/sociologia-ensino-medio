## 🎯 Como Usar Este Sistema

### Para Começar (Hoje)
1. **Leia RESUMOS_EXECUTIVOS_TOPICOS.md**
   - Escolha 1-2 tópicos que mais interessa
   - Veja conceitos-chave rápidos
   - Identifique autores principais

### Para Pesquisar (Próximas Semanas)
2. **Use GUIA_PESQUISA_CONTEUDO.md**
   - Vá para a seção do tópico escolhido
   - Consulte as fontes listadas
   - Tomar notas usando modelo de nota de leitura
   - Compile referências bibliográficas

### Para Planejar (Concomitantemente)
3. **Siga PLANO_ACAO.md**
   - Marque tarefas como "in_progress"
   - Respeite cronograma por tópico (ou adapte conforme necessário)
   - Use workflow recomendado (pesquisa → estruturação → escrita → revisão → implementação)
   - Acompanhe na tabela de status

### Para Escrever (2-3 semanas depois)
4. **Estruture conteúdo em HTML**
   - Use modelo HTML proposto em GUIA_PESQUISA_CONTEUDO.md
   - Adicione ao array `topicsData` no `index.html`
   - Inclua toggles para conceitos expandíveis
   - Use callout boxes para destaque
   - Sempre cite autores com datas
   - Finalizar com referências

### Para Validar (Ao fim)
5. **Verifique checklist de qualidade**
   - Pesquisa, escrita, referências, HTML, revisão
   - Teste no site (mobile e desktop)
   - Solicite feedback a educador/especialista se possível

## 🎓 Sobre Transposição Didática

### Princípios para Transformar Pesquisa em Aula

1. **Simplificar sem empobrecer**
   - Use linguagem acessível
   - Explique termos técnicos
   - Mantenha rigor conceitual

2. **Contextualizar realidade do estudante**
   - Conexão com vida cotidiana
   - Exemplos brasileiros concretos
   - Mostrar relevância prática

3. **Usar recursos diversificados**
   - Toggles para expandir conceitos
   - Callout boxes para destacar
   - Exemplos visuais/práticos
   - Citações de autores

4. **Promover pensamento crítico**
   - Questões desafiadoras
   - Múltiplas perspectivas
   - Análise de contradições

5. **Indicar aprofundamento**
   - Referências para leitura adicional
   - Autores primários
   - Pesquisas académicas
